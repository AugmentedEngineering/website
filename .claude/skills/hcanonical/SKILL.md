# Html Canonical

When generating code using html-canonical, always check:

1. **Semantic and accessible**: The structure generated must be accessible and semantic, easily compatible with screen readers.
2. **Lean**: Implement a lean structure, avoid unnecesary nesting or superfluous markup.
3. **Content models**: Always rely on the content model as described in the html spec.
